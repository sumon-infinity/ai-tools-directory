## ✨ Features

- 🔍 **Live Search** — Instantly filter tools by name, description, or category
- 🏢 **11 Departments** — Browse by Sales, Marketing, Finance, HR, Legal, IT, and more
- 🏷️ **Category Pills** — Quick filter by sub-category within each department
- 🌙 **Dark Theme** — Sleek dark UI with green accents
- 📱 **Fully Responsive** — Works on desktop, tablet, and mobile
- ⚡ **Zero Dependencies** — Pure HTML, CSS, and JavaScript — no build tools needed

## 🚀 Getting Started

### Option 1 — View Online
Simply visit the **[live GitHub Pages site](https://your-username.github.io/ai-tools-directory/)**.

### Option 2 — Run Locally
No build steps needed. Just clone and open:

```bash
git clone https://github.com/your-username/ai-tools-directory.git
cd ai-tools-directory
# Open index.html in your browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

## 📖 How to Use

1. **Search** — Type any tool name or keyword in the search bar
2. **Filter by Department** — Click a department in the left sidebar
3. **Filter by Category** — Use the category pills to narrow down further
4. **Visit a Tool** — Click the **VISIT →** button on any card to open the tool's website

## 🤝 Contributing

Contributions are welcome! If you'd like to add a new AI tool or fix something:

1. Fork this repository
2. Open `index.html` and find the `allTools` array
3. Add your tool in this format:
```javascript
{
  name: "Tool Name",
  url: "https://toolwebsite.com",
  desc: "Short description of what the tool does.",
  tags: ["Department"],
  category: "Sub-Category"
}
```
4. Submit a Pull Request
